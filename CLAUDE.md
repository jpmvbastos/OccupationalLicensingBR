# Project Instructions

1. Use `/stata` commands and skills for Stata work. Convert PDFs to Markdown with the `/pdf` skill before reading them.

2. Always delegate jobs to agents, run background tasks with `run_in_background: true`, and keep the main window clear of raw output — synthesize and present clean summaries.

3. Always work inside a git worktree (`git worktree add -b <branch> <path>`). Never implement directly on `main`.
